pg 的数据类型以及连接方式

第一天学习 pgSQL

用自己想记录东西的方式书写一下了解到的事情

数据类型 描述
char 不可变长度字符串
varchar 可变长度字符串
int 整型
float 浮点型
numeric 精确值（？）

连接命令：psql
其支持上下键翻阅命令行，自动补全
常用的参数如下：

-h : host 数据库地址 
-p : post 端口号
-U : 用户名
-d : 数据库名称
连接时如果带上 -E  即可在敲击元命令时获得 sql 语句

常用的命令：

元命令：
\d : 查看当前数据库中所有的表
   其后可接入 表名，用来查看表的数据结构
\db : 查看表空间信息
\dn : 查看所有模式信息
\du \dg : 查看数据库所有用户和角色
\x : 设置查询结果的输出模式,切换输出方式
\? : 相当于 help 
\timing : 是否显示 sql 的执行时间的开关
